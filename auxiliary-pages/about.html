ABOUT

This is a narrative story with multiple endings created by me, Tassia Accioly, as a one week and a half project.

The first week the game was written a bit everyday. The total game has about 12k words. The second week was the actual building of the game with writing all the HTML, CSS and Javascript required.

I'm brazilian and have a bachelor's degree in Movie Making. Because I love of all things interactive, I decided to branch out to web dev and game dev to turn my movies into interactive experiences in the future.

The thing I love the most is learning, and if I don't know how to do something I'll definitely find out how to do it.
